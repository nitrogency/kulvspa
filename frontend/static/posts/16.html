Ubuntu - creating unattended/automatic .ISO's

A guide to creating unattended/automatic Ubuntu Server and Ubuntu Desktop .ISO files

Author: nitrog

Ever had a situation where you wanted to create an automatic, pre-configured Ubuntu .ISO file that you can just mount, boot and the rest gets done on it's own? With this guide, we'll learn how to do just that.

Requirements:

  • Internet connection
  • Ubuntu autoinstall script
